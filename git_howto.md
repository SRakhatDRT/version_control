# Подсказка по GIT

*Инициализация репозитория:*
```sh
git init
```
*Статус репозитория:*
```sh
git status
```
*Добавление файла:*
```sh
git add
```
*Зафиксировать изменения:*
```sh
git commit -m <message>
```
*Передача содержимого из удаленного в локальный репозиторий:*
```sh
git pull
```
*Передача изменения из локального репозитория в удаленный:*
```sh
git push
```
